A class to browse tree structures. When creating the object, you have one required attribute and three optional attributes: node (the node where you wish to start browsing, usually the root of the tree)

parent (the name of the method used to get to the parent)

children (the name of the method used to get to the children)

properties (the name of the method used to get to properties)
